# Projeto de Automação de Testes utilizando Cypress

## 🛠️ Pré-requisitos
* Node.js
* VS Code (ou outro editor de sua preferência)

---

## 🚀 Como Executar o Projeto

1. **Baixar o projeto:** Clone ou baixe o projeto através do repositório.
2. **Instalar as dependências:** Abra o terminal na pasta do projeto e execute o comando:
   ```bash
   npm install
3. **Executar os testes: Para abrir a interface do Cypress e rodar os testes, utilize o comando:
   ```bash
   npx cypress open
